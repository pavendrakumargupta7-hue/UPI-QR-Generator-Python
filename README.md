# UPI-QR-Generator-Python
"UPI QR-Code Generator" using Python This project generates UPI payment QR codes based on user input. The user enters a UPI ID, name, amount, and message, and the program creates a scannable QR code image. The code supports Google Pay, PhonePe, and Paytm. Built using Python, qrcode, and Pillow libraries for college project use.
